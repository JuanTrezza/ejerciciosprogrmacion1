# Ejercicios de Programacion

Este repositorio reune practicas y ejercicios de Python organizados por unidad y por bloques de trabajo.
La idea es que el contenido se pueda revisar de forma clara, tanto leyendo el codigo como ejecutando los archivos que ya tienen resolucion.

## Contenido actual del repositorio

### Ejercicios generales

- `ejercicios_inicial.py`: ejercicios basicos resueltos.
- `ejercicios_avanzado.py`: ejercicios de mayor dificultad resueltos.
- `practica_unidad02.py`: practica complementaria de la Unidad 2.
- `practica_unidad03.py`: practica complementaria de la Unidad 3.

### Unidades

- `unidad2.py`: ejercicios de la Unidad 2.
- `unidad3.py`: ejercicios de la Unidad 3.
- `unidad4.py`: ejercicios de la Unidad 4.
- `unidad5.py`: ejercicios de la Unidad 5.
- `unidad6.py`: ejercicios con menu interactivo para ejecutar opciones desde consola.
- `unidad7.py`: ejercicios sobre listas, con consignas y resoluciones.
- `unidad8.py`: guia de ejercicios sobre tuplas y diccionarios. Este archivo esta preparado como guia de practica con consignas.

### Simulacion de parcial

Dentro de la carpeta `simulacion_parcial_01/simulacion_parcial_01/` se agrego una practica de simulacion de parcial con los siguientes archivos:

- `ejercicio_01.py`: validar contrasena.
- `ejercicio_02.py`: combinar dos listas.
- `ejercicio_03.py`: eliminar duplicados.
- `ejercicio_04.py`: contar apariciones.
- `ejercicio_05.py`: numero primo.
- `ejercicio_06.py`: conversor de unidades con menu interactivo.
- `conversiones.py`: modulo auxiliar para las conversiones del ejercicio 6.
- `Importante.md`: recomendaciones generales para encarar los ejercicios.
- `README.py`: guia de practica en formato Python valido.
- `masEjercicios.py`: archivo de apoyo con guia y ejercicios adicionales.

## Como ejecutar los archivos

Desde la raiz del proyecto se pueden ejecutar los archivos Python directamente.

### Ejemplos

```bash
python ejercicios_inicial.py
python ejercicios_avanzado.py
python unidad6.py
python unidad7.py
python unidad8.py
python simulacion_parcial_01/simulacion_parcial_01/ejercicio_06.py
```

## Que se puede revisar facilmente

- Resoluciones ya implementadas en los archivos principales.
- Ejercicios organizados por unidad.
- Simulacion de parcial con ejercicios separados por archivo.
- Guias de practica para leer consignas sin necesidad de resolver todo dentro de un unico archivo.

## Objetivo del repositorio

El objetivo es que el profesor pueda revisar de forma ordenada los ejercicios realizados, viendo tanto la organizacion del trabajo como las resoluciones desarrolladas en Python.

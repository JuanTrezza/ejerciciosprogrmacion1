# Ejercicios de Programacion

Este repositorio contiene ejercicios de Python organizados por nivel y por unidad.
Algunos archivos estan preparados como guias con consignas y respuestas, y otros como practicas interactivas para ejecutar desde consola.

## Estructura del repositorio

- `ejercicios_inicial.md`: guia en formato Markdown con consignas de ejercicios basicos.
- `ejercicios_inicial.py`: ejercicios iniciales en formato `Consigna + Respuesta`, con una demostracion simple.
- `ejercicios_avanzado.md`: guia en formato Markdown con consignas de ejercicios avanzados.
- `ejercicios_avanzado.py`: ejercicios avanzados en formato `Consigna + Respuesta`, con demostracion para correccion.
- `unidad2.py`: ejercicios correspondientes a la Unidad 2.
- `unidad3.py`: ejercicios correspondientes a la Unidad 3.
- `unidad4.py`: ejercicios correspondientes a la Unidad 4.
- `unidad5.py`: ejercicios correspondientes a la Unidad 5.
- `unidad6.py`: ejercicios de la Unidad 6 con menu interactivo para ejecutar cada punto por separado.

## Como revisar los ejercicios

### Archivos con demostracion

Estos archivos muestran resultados de ejemplo al ejecutarse:

```bash
python ejercicios_inicial.py
python ejercicios_avanzado.py
```

### Archivo interactivo de la Unidad 6

La Unidad 6 permite elegir un ejercicio desde un menu y probarlo manualmente:

```bash
python unidad6.py
```

### Otras unidades

Tambien se pueden ejecutar directamente las otras unidades segun corresponda:

```bash
python unidad2.py
python unidad3.py
python unidad4.py
python unidad5.py
```

## Objetivo

El objetivo del repositorio es que cada ejercicio quede presentado de manera clara, ordenada y facil de corregir, permitiendo al profesor revisar tanto las consignas como el funcionamiento de cada resolucion.

# Ejercicios de Programacion

Este repositorio contiene ejercicios resueltos de clase organizados por unidad.

## Estructura

- `unidad2.py`: ejercicios de la Unidad 2 (1 al 10).
- `unidad3.py`: ejercicios de la Unidad 3 (1 al 17).
- `unidad4.py`: ejercicios de la Unidad 4 (1 al 8).

## Como usar

1. Ejecuta el archivo de la unidad que quieras revisar.
2. Elige un numero de ejercicio en el menu.
3. El programa muestra la resolucion correspondiente.
4. Ingresa los datos pedidos para verificar cada ejercicio.

Ejemplo:

```bash
python unidad3.py
```

Tambien puedes revisar la Unidad 4 con:

```bash
python unidad4.py
```

## Objetivo

Que cada ejercicio quede identificado de forma clara para facilitar la correccion, revision docente y verificacion manual de resultados.
